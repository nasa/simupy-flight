NESC Test Cases

These are implementations of the NESC test cases using the SimuPy Flight Vehicle Toolkit.

Additional information about the cases can be found at the NASA Engineering and Safety Center’s “Six degree-of-freedom (6-DOF) Flight Simulation Test Cases.”

Download all examples in Python source code: nesc_test_cases_python.zip

Download all examples in Jupyter notebooks: nesc_test_cases_jupyter.zip
